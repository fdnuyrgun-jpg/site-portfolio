# EduVantage — Платформа Твоего Будущего

Современный, высококонверсионный лендинг для образовательной IT-платформы. Проект разработан с использованием передовых технологий фронтенд-разработки для обеспечения максимальной скорости загрузки и отличного пользовательского опыта.

## Особенности проекта

- **Полностью адаптивный дизайн**: Корректное отображение на смартфонах, планшетах и десктопах.
- **Интерактивные компоненты**: Модальные окна, аккордеоны (FAQ), кастомные селекторы и анимированные элементы.
- **Производительность**: Использование современных методов оптимизации и легких библиотек.
- **Aesthetic UI**: Чистый код с использованием Tailwind CSS для быстрой кастомизации стилей.

## Стек технологий

- **React 19**: Мощная библиотека для построения интерфейсов.
- **Tailwind CSS**: Utility-first CSS фреймворк для стилизации.
- **Lucide React**: Набор красивых и легких иконок.
- **Vite**: Быстрый инструмент сборки для современной веб-разработки.

## Запуск проекта локально

### Предварительные требования
Убедитесь, что у вас установлен [Node.js](https://nodejs.org/).

### Шаги по установке:

1. **Установите зависимости**:
   ```bash
   npm install
   ```

2. **Запустите сервер разработки**:
   ```bash
   npm run dev
   ```

3. **Откройте приложение**:
   Перейдите по адресу `http://localhost:3000` в вашем браузере.

## Структура проекта

- `components/`: Переиспользуемые React-компоненты (Навигация, Герой-секция, Формы и т.д.).
- `constants.tsx`: Конфигурационные данные лендинга (тексты программ, цены, отзывы).
- `types.ts`: Типизация TypeScript для обеспечения стабильности кода.
- `App.tsx`: Главный компонент, собирающий структуру страницы.
